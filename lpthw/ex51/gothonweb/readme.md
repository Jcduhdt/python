# 创建布局模板
> 每次都写一个完整的网页很烦  
> 这时候就需要有一个不会变的大框架，以后只要把内容填进去就行  
> 创建布局模板 就是提供了通用的头文件和脚注的外壳模板，减少重复动作  
> layout.html就是这个外壳模板  
> index_laid_out.html 和 hello_form_laid_out.html 就是去掉样板代码的代码  
> 开头执行的语句就是把这两个模板内容传递到layout.html
